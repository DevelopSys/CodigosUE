Driver: 	libreria que permite la "traduccion" entre java y la base de datos

Statement: 	Permite ejecutar querys sobre una connection. Lo malo que tiene es que la query es "plana". Ademas permite un ataque de SQL Injection
PrepareStatement:	Permite ejecutar querys sobre una connection parametrizadas. Evita SQL Injection

		execute() -> bool
		executeUpdate() ->int

ResultSet: 	La tupla de datos de la query de seleccion
